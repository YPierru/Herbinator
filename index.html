Herbinator

Ce programme est le résultat de notre projet de fin de deuxième année de DUT. Nous étions 3 étudiants : Fabio, Evelyne et Yan.

Pour lancer le programme, télécharger "Herbinator_runnable.jar", ouvrez une console et tapez java -jar Herbinator_runnable.jar

Présentation

Générale

Herbinator est un tableau graphique intéractif en java, utilisant le capteur de mouvement de la Kinect. Il se base sur le travail du peintre Auguste Herbin, qui avait créé un alphabet graphique : chaque lettre correspond à une série de formes d'une couleur précise.

Pour créer un tableau sur Herbinator, il faut rentrer un mot, et le programme générera un tableau en utilisant l'alphabet graphique.

Intérêt de la Kinect

Actuellement la partie Kinect est désactivée. Une procédure permettant d'utiliser la Kinect (installation des dépendances) sera disponible prochainement.

Herbinator contient un item "Activer la kinect", qui permet de capter la main d'une personne devant la kinect, et cette main devient le pointeur. L'utilisateur peut alors effectuer différentes actions avec sa main :

  • Déplacer une forme : On place sa main sur une forme et on "prend" la forme, c'est à dire qu'on recule sa main. Le pointeur change de couleur lorsque la forme est bien attrapée. Pour relâcher la forme, il suffit de pousser sa main.
  • Changer la couleur d'une forme : Il faut pousser sa main en étant sur une forme. La couleur sera choisie aléatoirement.

Fonctionnalités

Générer un tableau

Aller dans Edition->Herbinisation

Dans cette fenêtre vous pouvez :

  • Choisir la taille du tableau
  • Taper le mot à traduire
  • Choisir le mode : Libre, vous pourrez déplacer les formes comme vous le souhaitez ou Pro, vous interchanger les formes entre elles
  • Plein écran : pour un affichage en plein écran (sans bordure)

Cliquer sur générer (exemple de génération en 600x400, "bonjour")

Vous pouvez déplacer des formes en "drag&drop". Vous pouvez aussi changer de couleur les formes en faisant un clic-droit sur la forme.

Sauvegarder votre tableau (.herbinator)

Allez dans Fichier->Sauvegarder (.herbinator)

Le .herbinator garde les propriétés de votre tableau. Si vous ouvrez un .herbinator sauvegardé précédemment, vous pourrez à nouveau déplacer des formes, les changer de couleur etc.

Sauvegarder en PNG

Allez dans Fichier->Sauvegarder (PNG) Vous sauvegarderez ainsi votre tableau en png. Contrairement au .herbinator, vous ne pouvez pas ouvrir le tableau sauvegardé en .png.

Régénérer le tableau

Allez dans Edition->Regenerer (même paramètres)

Le programme générera un nouveau tableau en gardant les paramètres saisies initialement (taille/mot/mode).

Activer la Kinect

Allez dans Edition->Activer la Kinect

Une fenêtre se lancera, montrant "ce que voit" la Kinect. Il faudra déplacer sa main devant la Kinect afin qu'elle "capte" votre main, et que le pointeur apparaisse sur Herbinator.

Quitter le fullscreen

Allez dans Edition->Quitter le fullscreen

Lorsque vous êtes en mode plein écran, cette fonctionnalité vous permet de faire réapparaitre la barre de menus.

Cacher la souris

Allez dans Edition->Cacher la souris

Cela rend le pointeur de la souris invisible. Utile lorsque la Kinect est activée.
